================================================================================
                        WHY THIS IS HAPPENING
================================================================================

Modern browsers block local files from loading other local files (CORS policy).

The solution is to serve the files over HTTP instead of file://.

That's what run-server.bat does - it starts a tiny web server on your computer.

================================================================================
                          ALTERNATIVE METHODS
================================================================================

If run-server.bat doesn't work:

METHOD 1: Manual Command Prompt
  1. Open Command Prompt
  2. Type: cd D:\Custom_Web_app
  3. Type: python -m http.server 8000
  4. Open browser to http://localhost:8000

METHOD 2: PowerShell
  1. Open PowerShell
  2. Type: cd D:\Custom_Web_app
  3. Type: .\run-server.ps1
  4. Open browser to http://localhost:8000

METHOD 3: VS Code
  1. Open VS Code
  2. Open folder: D:\Custom_Web_app
  3. Right-click index.html
  4. Select "Open with Live Server"
  5. Browser opens automatically

METHOD 4: Chrome Extension
  1. Install "Web Server for Chrome" extension
  2. Open extension
  3. Choose folder: D:\Custom_Web_app
  4. Click link (http://127.0.0.1:8000)
  5. Browser opens with app
