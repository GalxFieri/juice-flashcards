## âš¡ TL;DR - 8 Steps to Live App

1. `npm install -g firebase-tools`
2. `firebase login`
3. Create project in Firebase Console
4. Enable 3 services in Console (Authentication, Firestore, Hosting)
5. `firebase init` (in your project folder)
6. Update `firestore.rules` and `firebase.json`
7. `firebase deploy`
8. Visit your app URL

**Total time: ~25 minutes**
